# Website Errors

## Payment Processing Errors

### "Payment Processing Failed"
**Possible Causes:**
- Insufficient funds
- Card declined by bank
- Incorrect card information
- Security holds by bank
- Technical issue with payment gateway

**Troubleshooting Steps:**
1. Verify card information (number, expiration, CVV)
2. Check with bank for declined transactions
3. Try different payment method
4. Clear browser cache and cookies
5. Try different browser
6. Contact support for manual order processing

### "Payment Timeout"
- Slow internet connection
- Payment gateway temporarily unavailable
- Session expired (return to cart and retry)

**Immediate Solution:**
- Offer manual order processing
- Reserve items at current price for 24 hours
- Apply same promotional codes manually

## Shopping Cart Issues

### Items Disappearing from Cart
- Cart contents saved for 7 days
- May be removed if out of stock
- Check "Saved for Later" section
- Clear browser cookies may reset cart

### Promotional Codes Not Working

**"Code Invalid" Error:**
- Code may be expired
- Code case-sensitive (try all caps)
- Minimum purchase requirement not met
- Code restricted to specific categories
- Code limited to one per customer
- Already used this code before

**"Code Cannot Be Combined" Error:**
- Only one promotional code allowed per order
- Code cannot be combined with current sale prices
- Offer to manually apply best discount

## Browse/Search Issues

### Items Not Appearing in Search
- Check spelling
- Try broader search terms
- Use filters to narrow results
- Item may be out of stock (check "Include Out of Stock" option)

### Images Not Loading
- Slow internet connection
- Browser cache issue (clear cache)
- Ad blocker interference
- Try different browser

## Checkout Errors

### "Address Validation Failed"
- Use standard USPS format
- Apartment/suite number in correct field
- PO Box not accepted for some items
- International addresses not supported

### "Order Cannot Be Submitted"
- Session timeout (refresh and retry)
- Out of stock item in cart
- Payment method declined
- Shipping restrictions for item
- Check error messages at top of page

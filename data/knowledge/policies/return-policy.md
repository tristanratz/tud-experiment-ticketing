# Return Policy

## General Return Guidelines

All items can be returned within **30 days** of purchase for a full refund, provided they meet the following conditions:

- Item is in original condition
- Original tags are still attached
- Receipt or proof of purchase is provided
- Item has not been worn, used, or damaged

## Exceptions

### Non-Returnable Items
- Opened software or digital products
- Personal care items (for hygiene reasons)
- Custom or personalized items
- Items marked as "Final Sale"

### Extended Return Windows
- Holiday purchases (Nov 1 - Dec 31): 60-day return window
- Defective items: Can be returned anytime within warranty period

## Return Process

1. Log into your account
2. Go to Order History
3. Select the order and click "Return Item"
4. Choose reason for return
5. Print prepaid return label
6. Drop off at any postal location

## Refund Timeline

- Refunds processed within 5-7 business days of receiving returned item
- Original payment method will be credited
- Shipping costs are non-refundable (except for defective items)

# Account Issues

## Login Problems

### Cannot Login - "Invalid Credentials"

**Common Causes:**
- Incorrect password (passwords are **case-sensitive**)
- Account not yet verified (check email for verification link)
- Account locked due to multiple failed attempts

**Solutions:**
1. Try password reset using "Forgot Password" link
2. Check spam folder for verification email
3. Wait 30 minutes if account is locked, then try again
4. Contact support if issue persists

> **Important:** After 5 failed login attempts, your account will be automatically locked for security purposes. Please wait 30 minutes before trying again.

---

### Account Locked

Accounts are automatically locked after **5 failed login attempts** within 15 minutes.

**Unlock Process:**
1. Wait 30 minutes for automatic unlock, OR
2. Use "Forgot Password" to reset immediately, OR
3. Contact support for immediate manual unlock (requires identity verification)

**Identity Verification Required:**
- Billing address on file
- Last 4 digits of payment method
- Recent order number

---

## Password Requirements

New passwords must meet the following criteria:

| Requirement | Details |
|------------|---------|
| Length | Minimum 8 characters |
| Uppercase | At least one uppercase letter (A-Z) |
| Lowercase | At least one lowercase letter (a-z) |
| Numbers | At least one number (0-9) |
| Special Characters | At least one symbol (!@#$%^&*) |

**Example of a strong password:**
```
MyP@ssw0rd2024!
```

## Account Creation Issues

### Email Already in Use

If you receive an "email already in use" error:

1. **Check if you already have an account** - Try the "Forgot Password" option first
2. **Verify the email address** - Check for typos in your email
3. **Look for old accounts** - You may have created an account previously
4. **Contact support** - Previous accounts can be merged or recovered

> **Tip:** Before creating a new account, try logging in with the "Forgot Password" option to see if an account already exists.

### Email Not Received

**Troubleshooting steps:**

1. Check spam/junk folders
2. Add `noreply@company.com` to your contacts
3. Wait up to **10 minutes** for email delivery
4. Use "Resend Verification Email" option in your account
5. Try an alternative email address if issue persists

**Common email providers and issues:**

- **Gmail** - May categorize as "Promotions" or "Updates"
- **Outlook/Hotmail** - Check "Other" inbox folder
- **Yahoo** - May require whitelisting our domain
- **Corporate emails** - May be blocked by company filters

## Profile Updates

### Cannot Update Address

**Required fields for address updates:**
- Full name
- Street address (line 1)
- City
- State/Province
- ZIP/Postal code
- Country
- Phone number

> **Note:** PO Boxes are not accepted for shipping addresses. Please provide a physical street address.

**Common issues:**

| Issue | Solution |
|-------|----------|
| ZIP code error | Verify zip code matches city/state |
| Invalid format | Use standard address format without special characters |
| PO Box rejected | Provide physical street address for shipping |
| International address | Ensure country is selected correctly |

### Cannot Update Payment Method

**Payment card requirements:**
- Card must not be expired
- Billing address must match card statement
- CVV (security code) required for new cards
- Some prepaid cards not accepted

**Supported payment methods:**

- ✓ Visa
- ✓ Mastercard
- ✓ American Express
- ✓ Discover
- ✓ PayPal
- ✓ Apple Pay
- ✓ Google Pay

## Two-Factor Authentication (2FA)

### Enabling 2FA

Two-factor authentication adds an extra layer of security to your account.

**Setup steps:**
1. Go to Account Settings → Security
2. Click "Enable Two-Factor Authentication"
3. Choose your method:
   - SMS (text message)
   - Authenticator app (recommended)
   - Email
4. Follow the setup wizard
5. Save backup codes in a secure location

> **Security Recommendation:** We strongly recommend using an authenticator app (like Google Authenticator or Authy) for the most secure 2FA experience.

### Lost Access to 2FA

If you've lost access to your 2FA method:

1. Use one of your backup codes (if saved)
2. Contact support with:
   - Account email address
   - Recent order details
   - Billing information for verification
3. Complete identity verification process
4. Support will disable 2FA temporarily
5. Set up new 2FA method immediately after regaining access
